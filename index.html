风琳 Player

基于 HarmonyOS NEXT 的第三方 Synology DS Audio（Audio Station）客户端：连接您的群晖 NAS，在本地浏览与播放音乐库。

应用定位

风琳 Player 不提供任何在线曲库、版权音乐内容或下载服务。您听到的曲目与封面均来自您本人拥有合法访问权限的 群晖 DSM / Audio Station 媒体索引与存储；本应用仅通过官方 API 拉取列表、元数据与流媒体地址。

Synology、DSM、Audio Station 等为群晖科技有限公司商标；本应用非群晖官方产品，与群晖无隶属关系。

主要能力（与当前版本功能一致）

  • 连接：QuickConnect 短 ID 与域名 / IP 自动分流、HTTPS、DSM 二步验证（OTP/TOTP）；登录信息本地安全保存。
  • 浏览与播放：歌曲 / 专辑 / 歌手 / 文件夹、全局搜索、「我最喜欢」（5 星）、NAS 歌单；AVPlayer 播放、系统 AVSession（通知栏 / 锁屏 / 车机卡片等）、后台音频。
  • 体验：迷你播放条、全屏正在播放与歌词（内嵌 / NAS、可关在线歌词）、可选无损与转码策略、封面与歌曲流本地缓存及上限管理。
  • 统计：本机播放历史与「最常 / 最近播放」；可选与 NAS 用户目录下 风琳Player/play_history.json 双向同步（需开启同步并保存 DSM 凭据以实现静默重登）。
  • 其他：智感握姿优化迷你条布局；会员权益通过华为应用内支付（IAP）在应用内订阅页购买与恢复。
运行平台：HarmonyOS（API 与工程配置以应用发布说明为准） 包名：cn.lvshin.music
