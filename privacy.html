风琳 Player 隐私政策

更新日期：2026年4月28日
生效日期：2026年4月28日

欢迎使用 风琳 Player（包名 cn.lvshin.music）。

风琳 Player 是运行在 HarmonyOS NEXT 上的群晖 Synology Audio Station（DS Audio）第三方客户端， 通过 Synology 官方 Web API 连接您自行部署的 DSM，在设备上浏览、播放与缓存来自您 NAS 的音乐元数据与媒体流。 我们非常重视您的隐私与个人信息保护。本隐私政策将说明我们如何收集、使用、存储、保护和管理您的信息。

请你在使用本 App 前，仔细阅读并充分理解本隐私政策。若你点击“同意”或继续使用本 App， 即表示你已阅读、理解并同意本隐私政策的内容。

一、特别说明：本 App 不提供歌曲内容

风琳 Player 通过 Audio Station 相关 API 访问您 NAS 上的媒体索引，不提供任何自有曲库、在线点播或音乐下载服务； 音频、封面、歌词等内容均来自您的群晖与您的合法授权配置。

本 App 不内置商业曲库，不向公网分发或售卖音乐；您在本 App 中的浏览与播放行为，数据流在您的客户端与您的 NAS（及您允许的网络路径）之间进行。

您应确保对访问内容拥有合法权利；Synology、DSM、Audio Station 等为群晖商标，本应用非群晖官方产品。

你应确保通过本 App 访问、播放或管理的音乐内容来源合法，并自行承担相应内容的权利义务。

二、我们如何收集和使用你的信息

为向您提供 DSM 登录、QuickConnect 解析、Audio Station 浏览与播放、可选 File Station 播放历史同步、会员与网络策略等功能， 我们可能处理以下信息（均与实现上述能力直接相关）。

1. NAS / DSM 连接与登录信息

当您配置连接时，本 App 可能在本地保存并用于请求 Synology API 的信息包括：

  • 您输入的 NAS 地址（含 QuickConnect 短 ID、域名或 IP）及解析后的 API 主机（含端口）；
  • 是否使用 HTTPS 等连接方式；
  • DSM 用户名与密码（或会话恢复所需的安全信息）；
  • 二步验证一次性码（OTP）仅在登录流程中按需使用，不持久化存储；
  • 会话标识（如 sid）等与官方 API 通信所需的令牌（仅存于客户端侧以维持会话）；
  • 用于 QuickConnect 服务发现的过程性网络请求（如访问 Synology 提供的解析与中继端点）。

上述信息仅用于与您指定的 DSM / Audio Station 通信；我们不会将您的 NAS 凭据上传至开发者自有服务器（应用功能不依赖此类上传）。 开启「与 NAS 同步播放历史」时，会使用已保存凭据与会话按需访问 File Station，在您 DSM 用户主目录下读写约定路径的文件（详见下文）。

2. 音乐库与媒体元数据

为展示列表与播放，本 App 通过 Audio Station 等接口获取并可能在内存或本地缓存中保存的包括：

  • 歌曲、专辑、歌手、文件夹、歌单等列表与标识；
  • 标题、艺人、专辑名、时长、路径或虚拟路径、星级与播放相关附加字段等；
  • 封面与流媒体的请求地址（由 DSM 生成，非本应用托管）；
  • 歌词文本（在您开启相关选项时，可能来自 NAS 歌词接口、本机歌词缓存或内嵌于已缓存音频流中的元数据）。

上述信息用于界面展示、搜索、播放与收藏写回 NAS；完整音频文件默认不主动上传至开发者服务器。 我们不会将您的曲库元数据用于广告营销或出售给第三方。

3. 播放记录、偏好与本地持久化

为提供续播、统计与体验优化，本 App 可能在本机应用沙箱内保存例如：

  • 当前队列、播放进度、播放模式、列表来源等（具体文件名以应用实现为准）；
  • 有效播放统计与「最常 / 最近播放」相关数据；
  • 音质与转码、歌词与缓存上限、会员相关偏好等设置；
  • 封面与歌曲流的磁盘缓存及索引（用于「本地歌曲」等能力）。

若您开启与 NAS 同步播放历史，应用会在您 DSM 用户主目录约定位置（如 风琳Player/play_history.json） 与您设备上的记录进行合并同步；该过程需要已保存的 DSM 凭据以便会话过期时静默续期，数据仍保存在您的 NAS 账户空间内。

「华为账号」端云同步播放历史等能力若将来上线，将在功能就绪时单独征得授权并说明；当前版本以本机与上述可选 NAS 同步为准。

4. 设备、网络与请求标识

为保障运行、计费或网络策略，本 App 可能使用：

  • 应用版本号：用于兼容逻辑与向您展示关于信息；发往 NAS 的 HTTP 请求可能携带标识为 FL_Player/<版本> 的 User-Agent（与实现对齐）。
  • 网络类型：用于判断是否为 Wi‑Fi / 以太等，以支撑「非 Wi‑Fi 自动转码」等会员能力与连接策略（不会在未说明的情况下单独上传至开发者服务器）。
  • 智感握姿：在支持设备上，我们可能使用握持手势相关系统能力微调迷你播放器布局方向，不用于识别个人身份。

除上述及系统或应用商店自带的诊断机制外，当前版本不内置向开发者后端批量上传崩溃遥测的商业化 SDK； 若您通过邮件等渠道主动联系并附上日志，则该部分内容由您自愿提供。

三、我们如何存储你的信息

本 App 中产生的数据可能存储在以下位置：

  1. 你的本地设备（应用沙箱）
    播放快照、播放历史统计、缓存、歌词与设置等；卸载应用后由系统规则清理沙箱数据。
  2. 你的群晖 DSM / Audio Station
    曲库元数据、星级、歌单等由 DSM 持有；本应用通过 API 读写您授权范围内的数据。
  3. 可选 NAS 文件（播放历史同步）
    若开启同步，在您用户主目录写入或更新 风琳Player/play_history.json，用于多终端合并统计，非向开发者公有云上传。

开发者不运营面向本应用的中央曲库或用户内容云；您的媒体文件不会因为使用本应用而默认上传到开发者服务器。

四、我们可能申请的系统权限（以当前应用清单为准）

以下与工程 module.json5 中声明的申请项一致；系统在授权界面展示的名称以实际设备与系统版本为准。

1. 联网权限（ohos.permission.INTERNET）

连接您的群晖 NAS / QuickConnect 解析服务、流媒体播放，以及通过内嵌网页打开常见问题与政策页面等。

2. 后台常驻（ohos.permission.KEEP_BACKGROUND_RUNNING）

与您在系统侧允许的后台音频播放能力配合，在切换应用或锁屏后尽量保持连续播放（具体行为受系统策略约束）。

3. 获取网络信息（ohos.permission.GET_NETWORK_INFO）

判断当前是否近似 Wi‑Fi / 以太等链路，以支撑应用内支付前网络状态提示，以及会员「非 Wi‑Fi 自动转码」等与网络类型相关的播放策略。

4. 手势/握持检测（ohos.permission.DETECT_GESTURE）

用于智感握姿：在支持的设备上感知左右手持握，微调底部迷你播放器布局（不影响您的音乐媒体内容本身）。

说明：媒体播控在通知栏、控制中心、锁屏等处的展示，通常由系统 AVSession 媒体会话负责，可能不单独以「通知权限」向您显示为可关项；蓝牙、车机等外接音频的路由与播控由系统及音频栈提供。本应用未在清单中单独申请「读写整盘用户相册/文件」类权限；封面与歌曲流缓存写入应用自有目录。

五、我们如何使用缓存

为了提升播放体验和加载速度，本 App 可能会在本地设备中保存部分缓存数据，例如：

  • 封面与歌曲流等媒体缓存（写入应用沙箱）；
  • 歌词文本等辅助缓存；
  • 流缓存索引（用于「本地歌曲」列表等与实现对齐的数据）。

封面与流缓存合计占用受您设置的「缓存上限」约束；超出时按文件最后修改时间从旧到新裁剪（与产品文档《常见问题》一致）。选择「无限制」时不在此机制下自动按上限删除。 您可在设置中执行「清理缓存」一次性清空相关目录（不删除 NAS 上的源文件）。

六、我们如何共享、转让或公开披露你的信息

我们不会出售、出租或非法共享你的个人信息。

在以下情况下，我们可能会共享必要信息：

  1. 获得您的明确同意后；
  2. 为实现您主动使用的功能所必需，例如您向 DSM 写入播放历史同步文件、通过华为 IAP 完成会员购买、您在反馈邮件中自愿提供日志等；
  3. 根据法律法规、司法机关或行政机关的要求；
  4. 为保护用户、我们或第三方的合法权益所必要。

除上述情形外，我们不会将您的 NAS 连接信息或曲库用于与说明目的无关的商业化共享。

七、第三方服务与处理者

当前版本未接入面向广告追踪的第三方 SDK；不会为投放广告将您的曲库或播放习惯出售给广告平台。

与实现功能相关的系统能力或外部服务主要包括：

  • Synology DSM / Audio Station / QuickConnect / File Station（经您配置的地址与账号）：由您在本地或远程网络中部署，本应用仅作为客户端访问；数据处理关系主要发生在您与群晖产品之间，请同时遵守群晖服务条款与隐私说明。
  • 华为应用内支付（IAP）：用于会员订阅、恢复购买等；订单与支付由华为应用市场 / AGC 流程处理，请以华为官方《应用内支付服务》等文档为准。
  • 内嵌网页（ArkWeb）：用于在应用内展示您配置的常见问题、本政策等 URL 页面；若指向第三方站点，请以该站点隐私政策为准。

若后续增加独立统计、崩溃上报、华为账号登录等能力，我们将在本政策中补充名称、用途与退出方式；实际以应用内说明与系统授权提示为准。

八、我们如何保护你的信息

我们会采取合理的安全措施保护你的信息，包括但不限于：

  • 尽量减少不必要的信息收集；
  • 对敏感配置信息进行本地安全存储；
  • 限制信息访问范围；
  • 防止未经授权的访问、泄露、篡改或丢失；
  • 持续优化 App 的安全性和稳定性。

但请你理解，互联网环境并非绝对安全。你也应妥善保管自己的 NAS 地址、账号、密码、访问令牌等信息，避免泄露给他人。

九、你如何管理自己的信息

你可以通过以下方式管理你的信息：

  1. 在 App 设置中修改或删除 NAS 连接配置；
  2. 清除本地播放记录、缓存、收藏或播放列表；
  3. 删除 App 本地数据；
  4. 在系统设置中关闭相关权限；
  5. 在 NAS 中自行删除由 App 生成或保存的数据；
  6. 卸载 App。

当你卸载 App 后，本地设备中由 App 保存的数据通常会被系统删除。但保存在 NAS 或其他外部存储中的数据，需要你自行管理和删除。

十、儿童个人信息保护

本 App 主要面向具备独立管理 NAS、音乐文件和本地存储能力的用户。

若你是未成年人，请在父母或其他监护人的陪同和指导下使用本 App。

未经监护人同意，未成年人不应自行配置 NAS 账号、上传反馈信息或进行会员购买等操作。

十一、账号注销与数据删除

当前版本不单独运营「风琳」用户体系；您可通过应用内退出登录清除 NAS 会话与本机敏感配置、清理本地数据或卸载应用以删除本机数据。

若将来提供华为账号端云同步或其它独立账号能力，我们将提供相应的注销、删除或关闭同步的途径，并在功能上线时更新本政策。
